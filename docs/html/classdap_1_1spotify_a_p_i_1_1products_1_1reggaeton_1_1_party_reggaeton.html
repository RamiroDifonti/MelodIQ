Detailed Description

Clase que representa un tipo específico de producto de Reggaeton, destinado a la creación de playlists de reggaeton para fiestas, utilizando un subgénero específico de reggaetón: "Reggaetón Dembow".

Crea una lista de canciones de reggaetón para fiesta usando la API de Spotify. Realiza una búsqueda por el subgénero "Reggaetón Dembow" y obtiene las canciones solicitadas.
