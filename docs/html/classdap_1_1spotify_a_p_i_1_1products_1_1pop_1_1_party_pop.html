Detailed Description

Clase que representa un producto de pop para fiestas.

Esta clase extiende PopProduct y sobrecarga el método create(SpotifyApi, int) para obtener una lista de canciones de pop específicamente para fiestas, utilizando los parámetros adecuados en la búsqueda a través de la API de Spotify.

Crea una lista de canciones de pop para fiestas.

Este método realiza una búsqueda en la API de Spotify utilizando el género "pop" y el estado de ánimo "party", y devuelve una lista de canciones obtenidas según la cantidad especificada.

Parameters
spotifyApiLa instancia de la API de Spotify utilizada para realizar la búsqueda de canciones.
amountEl número de canciones que se desean obtener.
Returns
Una lista de objetos Song que representan las canciones obtenidas.
