Detailed Description

Clase abstracta que representa un producto de reggaetón.

Esta clase sirve como base para los diferentes tipos de productos de reggaetón, como playlists o géneros específicos, y define el método create(SpotifyApi, int) que debe ser implementado por las clases hijas para crear listas de canciones de reggaetón.
