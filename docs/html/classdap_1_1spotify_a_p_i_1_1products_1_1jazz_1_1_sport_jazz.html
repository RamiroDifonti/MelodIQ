Detailed Description

Clase concreta que define el comportamiento de los productos de tipo Jazz para deporte. Esta clase extiende la clase JazzProduct y sobreescribe el método create(SpotifyApi, int) para crear una lista de canciones de jazz adecuadas para hacer deporte.
