Detailed Description

Clase concreta que define el comportamiento de los productos de tipo Pop para estudio en la aplicación.

Esta clase extiende de PopProduct y sobreescribe el método create(SpotifyApi, int) para crear una lista de canciones de pop relacionadas con el estado de ánimo de estudio.

Método que crea una lista de canciones de pop para estudiar.

Realiza una solicitud a la API de Spotify para obtener canciones de pop relacionadas con el estado de ánimo de estudio y las agrega a una lista de canciones de tipo Song.
