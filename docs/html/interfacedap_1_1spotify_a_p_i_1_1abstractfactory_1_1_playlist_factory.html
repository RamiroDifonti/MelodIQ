Detailed Description

Interfaz para la creación de productos de playlists de diferentes géneros musicales. Esta interfaz define los métodos para crear productos de tipo Pop, Jazz y Reggaeton. Las implementaciones concretas de esta interfaz, como PartyFactory, se encargan de generar los productos adecuados según el estilo deseado.
