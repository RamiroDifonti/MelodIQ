\doxysubsection{Detailed Description}
Clase principal de la interfaz gráfica de usuario (\doxylink{classdap_1_1spotify_a_p_i_1_1_g_u_i}{GUI}) que interactúa con la API de Spotify. Esta clase configura la autenticación a través de la API de Spotify utilizando las credenciales del cliente, obtiene un token de acceso y pasa dicho token a un objeto {\ttfamily \doxylink{}{Spotify\+Api}} para realizar futuras solicitudes.
