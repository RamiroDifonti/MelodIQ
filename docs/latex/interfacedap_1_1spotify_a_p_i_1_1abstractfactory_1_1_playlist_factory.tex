\doxysubsection{Detailed Description}
Interfaz para la creación de productos de playlists de diferentes géneros musicales. Esta interfaz define los métodos para crear productos de tipo Pop, Jazz y Reggaeton. Las implementaciones concretas de esta interfaz, como {\ttfamily \doxylink{classdap_1_1spotify_a_p_i_1_1abstractfactory_1_1_party_factory}{Party\+Factory}}, se encargan de generar los productos adecuados según el estilo deseado.
